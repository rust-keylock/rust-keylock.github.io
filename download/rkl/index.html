Download

rust-keylock is delivered as a desktop app, an Android app and a shell app.

Here is how you can install what you are interested in:

Desktop

The desktop application can currently run on Linux (x64) and Windows (x64).

Prerequisites

Java 8 (or later) should be installed on your system.

Installation and Usage

  1. Download the .zip or .tar.gz file.

    • Both compressed files contain the same application, so you may select whichever suits your environment.
    • You may optionally verify the integrity of the downloaded files using SHA1 or SHA512.

  2. Uncompress the downloaded file anywhere in your system

  3. Go to the directory where the files were decompressed and run

    java -jar desktop-ui-0.3.0.jar

Android

We hope that F-Droid will host the app shortly.

Until then, you may download the .apk from here and install it yourself.

Shell

Currently, the shell application is provided via cargo. Therefore, it can run where Rust can be installed.

Prerequisites

Rust should be installed on your system.

Installation and usage

  1. Install by issuing cargo install rust_keylock_shell.
  2. Run executing rust-keylock from the command line.
