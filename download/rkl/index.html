Download and Install

rust-keylock is delivered as a desktop app, an Android app and a shell app.

Here is how you can install what you are interested in:

Desktop

There are several choices for desktop users:

Snap

Install the rust-keylock-ui from the Snap Store

Arch User Repository

Arch users may find the application in AUR.

GitHub

Binaries may be doqnloaded from the project’s GitHub Repository. Since version 0.8.0, there are binaries for Linux and Windows.

Notes for running the GitHub binaries

  • Java 8 or higher should be instaled.

  • After downloading and untating / unzipping, just run the rust-keylock-ui executable.

Android

You may download the .apk from here and install it yourself on your devices.

Please note that the .apk is built and deployed automatically by Travis CI upon each release to ensure that the code being built is publicly available.

Shell

Currently, the shell application is provided via crates.io and cargo.

Prerequisites

Rust should be installed on your system.

Installation and usage

  1. Install by issuing cargo install rust_keylock_shell.
  2. Run executing rust-keylock from the command line.
